# Test cases and scripts for testing Randoop
## Overview
Randoop is a unit test generator for Java. See [Randoop GitHub repository](https://github.com/randoop/randoop) for details.

This repository contains test cases and scripts designed for testing Randoop, specifically for comparison with the results presented in the paper "GRT: Program-Analysis-Guided Random Testing." A copy of this paper is available in this directory.

## Usage Instructions
### Running Scripts
For instructions on using the scripts, refer to `reproinstructions.txt`.

### Building Tests
To obtain the sources to be tested, follow the guidelines in `tests/README.build`.
